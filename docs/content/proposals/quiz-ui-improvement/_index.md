# Quiz UI Improvement

Three cross-cutting enhancements to all quiz types (Standard, Reverse, Freeform).

1. **Override Answer** — Mark an incorrect answer as correct to handle OpenAI false positives
2. **Skip Word** — Permanently exclude a word from all future quizzes
3. **Next Review Date** — Show when a word will next appear and allow choosing the next day
