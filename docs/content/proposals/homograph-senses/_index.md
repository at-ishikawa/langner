# Homograph Senses

Give same-spelling / different-sense vocabulary entries (e.g. `record` the noun vs `record` the verb) an independent learning-log series each, by adding `part_of_speech` as a sense discriminator to the learning key — so their success/failure logs, spaced-repetition schedule, status, and analytics stop commingling, and the Relearn quiz shows the sense that was actually failed.
