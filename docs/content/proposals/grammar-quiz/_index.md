# Grammar Quiz

A quiz that turns your own journal writing into grammar practice. Journal entries are stored verbatim; grammar mistakes are annotated as spans inside them, each tagged with a free-form category. The quiz shows a sentence with the mistake highlighted, you type the correction, and an AI grader schedules it with the same spaced-repetition machinery the vocabulary quizzes use. Category tags make it possible to see which kinds of mistakes you make most.
